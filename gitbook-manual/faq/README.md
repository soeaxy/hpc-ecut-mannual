## 账号问题

### 第一种情况

进来的是登陆节点
![第一种情况](../assets/image-051.png)

这个时候，进来的是登陆节点，显示的ip是10.10.10.253
![第一种情况](../assets/image-052.png)

如果不使用容器实例，比如科学计算，用cpu那种，注意要切换到计算节点，命令是ssh admin。这样就到了计算节点。
![第一种情况](../assets/image-053.png)

### 第二种情况

如果是使用docker，要在容器实例里面执行，这里点ssh就到了容器实例里面。
![第二种情况](../assets/image-054.png)

注意此时的网址和账号
![第二种情况](../assets/image-055.png)

也可以使用命令 ssh 173.0.1.2
![第二种情况](../assets/image-056.png)

### 权限问题

进来之后，如果想安装python依赖，可能会权限不够，要将docker里面账号的权限提升到root，命令是sudo -s
![权限问题](../assets/image-057.png)
