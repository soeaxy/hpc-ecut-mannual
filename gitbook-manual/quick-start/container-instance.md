# 容器实例入门

本页用于快速理解容器实例的创建与启动流程。

## 容器实例

1. 登录后进入控制台，在页面菜单中，点击「容器实例」。
![容器实例](../assets/image-004.png)

1. 在容器实例页面，点击创建容器。
![容器实例](../assets/image-005.png)

1. 在容器创建页面：选择创建模式、名称、GPU 型号、GPU 数量、开发工具、开发镜像，最后创建即可。
![容器实例](../assets/image-006.png)

1. 创建完成后点击启动开机，进入命令行即可开始使用。
![容器实例](../assets/image-007.png)
